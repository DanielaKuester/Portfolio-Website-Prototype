Project 1: Animal Trading Card

The Animal Trading Card was the first little Udacity project that included HTML and CSS. In this project, I explored some basics of CSS like borders, the border-radius and shadows. I also discovered my love for linear gradients. I just had to include them in this Portfolio Website, too. So: Now you know where that preference comes from.

Project 2: Portfolio Website

The Portfolio Website was the first a bit more complex Udacity project with HTML and CSS. For this project, Udacity gave us a mockup of a portfolio and the task was to recreate it while giving it a personal touch. It's also the first project that involves responsive design and adapts to different screen widths (on PCs, tablets, phablets and smartphones).
